# Startup Dashboard using QlikView

## Overview

This project is a simple dashboard created using QlikView as part of an academic assignment. The main objective was to explore the QlikView software and understand how dashboards and visualizations are built.

## Features

* Basic startup data visualization
* Simple dashboard layout
* Interactive charts and filters

## Tools Used

* QlikView
* CSV/Excel dataset
